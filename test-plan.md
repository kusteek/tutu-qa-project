# План тестирования Tutu.ru

## Цель
Проверить функциональность сайта Tutu.ru: поиск билетов, формы ввода, фильтры, бронирование и адаптивность интерфейса.

## Область тестирования
- Поиск билетов (поезда, самолеты, автобусы)  
- Фильтры и сортировка результатов  
- Бронирование билетов  
- UI/UX и адаптивность  

## Типы тестирования
- Функциональное  
- UI/UX  
- Регрессионное  
- Автоматизация поиска билетов  

## Инструменты
- Браузеры Chrome/Firefox  
- GitHub и Markdown  
- Playwright (для автотестов)  

## Риски
- Некорректный ввод данных  
- Проблемы адаптивного интерфейса  
- Потеря данных при обновлении страницы
