# BUG-002

**Название:** Пользователь может выбрать дату возврата раньше даты туда

**Шаги:**
1. Открыть форму поиска  
2. Выбрать дату возврата раньше даты туда  
3. Нажать "Найти"

**Ожидаемый результат:**  
Появляется сообщение об ошибке с подсказкой выбрать корректную дату.

**Фактический результат:**  
Поиск выполняется, но результаты некорректны.

**Приоритет:** Средний
