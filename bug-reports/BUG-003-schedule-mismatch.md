# BUG-003

**Название:** Несоответствие расписания

**Шаги:**
1. Поиск "Москва → Ярославль"  
2. Сравнить расписание с официальным

**Ожидаемый результат:**  
Время отправления и прибытия соответствует официальному расписанию.

**Фактический результат:**  
Некоторые рейсы отображаются с неверным временем.

**Приоритет:** Средний
