# BUG-001

**Название:** Поиск билетов запускается без заполнения городов

**Шаги:**
1. Открыть Tutu.ru  
2. Оставить поля пустыми  
3. Нажать "Найти"

**Ожидаемый результат:**  
Система показывает сообщение о необходимости заполнить поля

**Фактический результат:**  
Поиск запускается, но результатов нет или страница пустая

**Приоритет:** Средний
