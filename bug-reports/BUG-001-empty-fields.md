# BUG-001

**Название:** Поиск билетов запускается без заполнения обязательных полей

**Среда (Environment):**  
- Браузер: Chrome 112  
- ОС: Windows 10  
- Устройство: ПК  

**Версия (Version):**  
- Tutu.ru v2.1 (по состоянию на 16.03.2026)  

**Описание:**  
При попытке поиска билетов без указания городов отправления и назначения сайт выполняет поиск, хотя должен уведомить пользователя о необходимости заполнения полей. Это может привести к пустым результатам и сбить пользователя с толку.  

**Шаги воспроизведения:**  
1. Открыть сайт [Tutu.ru](https://www.tutu.ru/).  
2. Оставить поля «Откуда» и «Куда» пустыми.  
3. Нажать кнопку «Найти».  

**Ожидаемый результат:**  
Система должна показать сообщение об ошибке: «Пожалуйста, заполните поля города отправления и назначения». Поиск не запускается.  

**Фактический результат:**  
Сайт выполняет поиск, но результаты пусты, нет уведомления об ошибке.  

**Приоритет:** Средний
