# Тест-кейсы Tutu.ru

| ID   | Название                         | Шаги                                                     | Ожидаемый результат |
|------|---------------------------------|----------------------------------------------------------|-------------------|
| TC01 | Поиск билетов с корректными данными | Ввести "Москва" → "Санкт-Петербург", выбрать даты, нажать "Найти" | Появляется список маршрутов |
| TC02 | Поиск с пустыми полями           | Поля “Откуда” и “Куда” пустые, нажать "Найти"          | Появляется сообщение о необходимости заполнить поля |
| TC03 | Некорректный город               | Ввести "123" в поле “Откуда”, нажать "Найти"           | Появляется сообщение о неверном формате |
| TC04 | Фильтрация по цене               | После поиска применить сортировку по цене              | Список отсортирован по цене |
| TC05 | Адаптивность на мобильных        | Открыть сайт на мобильном, проверить меню и кнопки     | Элементы корректно отображаются |
| TC06 | Некорректные даты                 | Дата возврата раньше даты туда, нажать "Найти"         | Появляется сообщение об ошибке |
| TC07 | Сохранение email при бронировании | Ввести email, перейти дальше и вернуться               | Email сохраняется |
