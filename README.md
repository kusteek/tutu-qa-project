# QA-проект: Tutu.ru

Тестирование сайта [Tutu.ru](https://www.tutu.ru/).  
В проекте реализованы: план тестирования, тест-кейсы, баг-репорты и пример автотеста на Playwright.

## Структура проекта
- `test-plan.md` — план тестирования  
- `test-cases/` — тест-кейсы  
- `bug-reports/` — баг-репорты  
- `automation-tests/` — автотесты  
- `screenshots/` — скриншоты багов

## Что я узнал
- Создание тест-планов и тест-кейсов  
- Оформление баг-репортов по стандарту QA  
- Автоматизация тестирования  
- Тестирование реального сайта
